# 今天吃什么 🍽️

一个帮助你决定"今天吃什么"的转盘决策应用！

## 功能特点

- 🎯 **转盘抽选**：点击按钮，转盘旋转，帮你决定吃什么
- 🏷️ **分类管理**：按面条、米饭、火锅等分类管理菜品
- ⏰ **场景切换**：早餐、午餐、晚餐、夜宵各有专属菜品池
- 🔥 **快速过滤**：不想吃辣？不想吃面？一键过滤
- ⚖️ **权重系统**：常吃的菜品更容易被抽到
- 💣 **踩雷标记**：标记踩雷菜品，下次可选择避开

## 使用方法

1. 直接在浏览器中打开 `index.html`
2. 选择场景（早餐/午餐/晚餐/夜宵/不限）
3. 点击「今天吃什么」按钮
4. 转盘旋转后显示结果
5. 可选择「再抽一次」或「就它了」

## 菜品管理

- 点击「管理菜品」按钮
- 可以添加自定义菜品
- 可以删除不需要的菜品
- 可以标记踩雷菜品

## 技术实现

- 纯前端实现，无需服务器
- 使用 Canvas 绘制转盘动画
- 使用 localStorage 持久化数据
- 响应式设计，支持手机和电脑

## 数据说明

所有数据存储在浏览器的 localStorage 中：
- 清除浏览器数据会丢失自定义菜品
- 不同浏览器之间数据不互通
